## About Me

Hi, I'm Arturo! I'm a Cal Poly SLO student studying Economics with a concentration in Quantitative Analysis, along with minors in Statistics, Computer Science, and Data Science. I enjoy making sense of all different types of data from different scenarios, turning numbers into insights that can help people. I'm especially interested in data analytics and data science, and I enjoy using data to answer questions, spot patterns, and support better decisions using tools like R, SQL, Python, pandas, and scikit-learn through my coursework and projects. I currently help evaluate student experiences in my role as an Assessment Assistant with the Cal Poly Scholars program, where I use data to identify trends and better understand student needs. I also apply these skills as an MBP Intern to support program initiatives and student-centered work.
